# Telegram Video Downloader Bot

Телеграм-бот для скачивания видео с YouTube, TikTok и Instagram. Отправь ссылку — получи видео.

## Возможности

- Скачивание видео с YouTube (включая Shorts), TikTok и Instagram (включая Reels)
- Лучшее доступное качество (YouTube — до 720p)
- Прогресс-бар загрузки
- Кнопка для получения описания видео
- Статистика скачиваний по платформам
- Лимит 10 скачиваний в сутки (для обычных пользователей)

## Стек

- Python 3.11
- pyTelegramBotAPI
- yt-dlp
- SQLite3
- ffmpeg

## Запуск

1. Установите зависимости
2. Добавьте `TELEGRAM_BOT_TOKEN` в переменные окружения
3. Запустите: `python src/bot.py`
